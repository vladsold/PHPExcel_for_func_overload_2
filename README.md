## How install in your project? 

Add into *composer.json* 
	
        "require": {
            "phpexcel/for_mbstring_func_overload_2": "dev-master"
        },
        "repositories": [
            {
                "type": "vcs",
                "url": "https://github.com/vladsold/PHPExcel_for_func_overload_2"
            }
        ]
        
Execute command: ```composer update```
